#################### For runnning Backend #####################
#1 Navigate to StudentManagementBackend folder Run composer update
#2 Create database & Update database credentials in .env DB_DATABASE, DB_USERNAME & DB_PASSWORD
#3 Run migration by using command php artisan migrate
#4 Run Application by using command php artisan serve
